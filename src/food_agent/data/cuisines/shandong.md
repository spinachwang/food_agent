# 鲁菜知识库

## 十大经典鲁菜

1. **糖醋鲤鱼** 🧂🧂 — 黄河鲤鱼，糖醋汁要亮，济南燕喜堂最有名
2. **九转大肠** 🧂🧂🧂 — 济南鲁菜代表，酸甜咸辣一菜五味
3. **葱烧海参** 🧂🧂 — 大葱只取章丘，胶东派经典
4. **油爆海螺** 🧂🧂 — 胶东派代表，火候秒级
5. **糖醋里脊** 🧂🧂 — 山东做法比南方更咸鲜
6. **爆炒腰花** 🧂🧂 — 大火爆，腰花脆嫩不出水
7. **四喜丸子** 🧂🧂 — 北方婚宴必有
8. **把子肉** 🧂🧂🧂 — 济南街头硬菜，肥而不腻
9. **油旋** — 济南老点心，层多酥脆
10. **孔府家宴** — 孔府菜精粹，礼仪与味并重

## 城市推荐

- **济南**
  - 大明湖周边（老字号扎堆，燕喜堂、聚丰德）
  - 芙蓉街（小吃汇集，把子肉、油旋）
  - 历下区（鲁菜正源，同和居、汇泉楼）
  - 鲁菜是八大菜系之首，宫廷菜之根

- **青岛 / 烟台（胶东）**
  - 青岛劈柴院（海鲜鲁菜，锅贴、烤鱿）
  - 烟台芝罘岛（胶东海鲜，鲁菜胶东派）
  - 威海（渔家菜，鲅鱼水饺、炸蛎黄）
  - 胶东菜偏清淡海鲜，与济南咸鲜有别

- **北京**
  - 前门 / 大栅栏（老字号鲁菜，丰泽园、同和居）
  - 什刹海（官府菜渊源）
  - 鲁菜曾是北京官府菜主流

- **曲阜（孔府菜）**
  - 孔府家宴（国宴级礼仪菜）
  - 阙里街（孔府菜馆子集中）
  - 仿古宴席需提前预定

## 咸度参考

- 🧂 — 淡口（清汤、清炒，沿海派系）
- 🧂🧂 — 中咸（北方家常，胶东菜多数）
- 🧂🧂🧂 — 偏咸（济南硬菜、下饭神器）
- 🧂🧂🧂🧂 — 极咸（卤味、腌制品，老济南口味）
- 🧂🧂🧂🧂🧂 — 卤咸（腊味、过咸菜，配粥用）

## 慎选情况

- 高血压 / 肾病 → 避咸菜、卤味、炖菜浓汤
- 痛风 / 嘌呤高 → 避海鲜老汤、内脏（大肠、腰花）
- 老人 / 小孩 → 推清淡派（胶东海参、清蒸鱼），避九转大肠
- 想清淡 / 减脂 → 鲁菜整体偏咸偏油，建议改推粤菜或江浙菜
- 海鲜过敏 → 避胶东派，主推济南派与孔府菜
