# 甜品饮品知识库

## 主流连锁速查

### 奶茶连锁（按价位排序）

1. **喜茶 HEYTEA** — 高端奶茶标杆（25-35 元），多肉葡萄 / 芝芝莓莓是招牌
2. **奈雪的茶** — 高端茶饮（25-35 元），霸气系列 + 软欧包
3. **瑞幸咖啡** — 性价比之王（9.9-25 元），生椰拿铁是国民饮料
4. **Manner Coffee** — 上海起家精品咖啡（15-30 元），澳白是爆款
5. **M Stand** — 上海网红咖啡（25-40 元），水泥系列是拍照神器
6. **星巴克** — 全球第一（25-45 元），拿铁 + 星冰乐
7. **CoCo 都可** — 经典老牌（10-18 元），奶茶三兄弟
8. **一点点** — 台湾血统（10-18 元），四季春 + 波霸
9. **蜜雪冰城** — 极致平价（3-8 元），冰鲜柠檬水是王者
10. **古茗 / 茶百道 / 沪上阿姨 / 书亦烧仙草** — 中端连锁（12-22 元）

### 糖水与烘焙

- **糖水店**：满记甜品、鲜芋仙（台式）、许留山（港式，关店中）、顺德双皮奶
- **烘焙连锁**：巴黎贝甜、面包新语、克莉丝汀、85°C、好利来（高端线：黑天鹅）
- **冰淇淋**：哈根达斯、DQ、Godiva、Bacio（胡同里意式冰淇淋）、野人牧坊
- **海底捞甜品站** — 火锅店内免费 / 平价甜品 + 网红打卡

## 城市推荐

- **上海**
  - 静安寺 / 武康路（% Arabica、Manner、M Stand 旗舰）
  - 人民广场（喜茶、奈雪旗舰店）
  - 浦东陆家嘴（星巴克烘焙工坊，全球最大）
  - 番禺路老法租界（甜品 + 咖啡独立小店密集）

- **北京**
  - 三里屯（% Arabica、Blue Bottle、M Stand）
  - 国贸 CBD（瑞幸密度全球第一）
  - 五道口（学生向，瑞幸 + 一点点扎堆）
  - 南锣鼓巷（甜品 + 糖水小店密集）

- **广州**
  - 天河城（喜茶 / 奈雪旗舰店）
  - 上下九（许留山、满记老牌糖水）
  - 北京路（顺德双皮奶总店分布）
  - 番禺市桥（糖水一条街，5-15 元平价糖水）

- **深圳**
  - 福田 CBD（喜茶总部、奈雪总部都这）
  - 南山科技园（瑞幸密度高）
  - 海岸城（% Arabica、Manner）

## 时段策略

- **早餐（6:00-10:00）**：咖啡（瑞幸、Manner、星巴克）+ 烘焙（巴黎贝甜、面包新语）
- **下午茶（14:00-17:00）**：15-17 点是高峰，避开；建议 14 点前或 17 点后
- **工作提神（10:00-12:00 / 15:00-17:00）**：咖啡时段，瑞幸 9.9 是打工人标配
- **餐后解腻（19:00-21:00）**：糖水（杨枝甘露、双皮奶）、冰淇淋
- **夜宵（21:00-02:00）**：24h 烘焙店、瑞幸部分门店、海底捞甜品站

## 慎选情况

- 糖尿病 / 减脂期 → 推无糖 / 少糖选项，瑞幸美式、一点点无糖、波霸茶
- 咖啡因敏感 → 推奶茶（低咖啡因）、糖水、烘焙，避免美式 / 拿铁 / 浓缩
- 睡前 4 小时内 → 推无咖啡因（糖水、烘焙、果汁）
- 商务正餐 / 宴请 → 推茶馆 / 酒廊 / 商务宴请套餐，甜品饮品太休闲
- 严重肠胃不适（胃酸过多） → 避开咖啡、浓茶、酸性果汁，推温和的米糊 / 燕麦奶
