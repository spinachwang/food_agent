# 小吃知识库

## 地域代表速查

1. **北京** — 卤煮火烧（北新桥）、豆汁（护国寺）、爆肚（爆肚冯）、炒肝（姚记）、艾窝窝、驴打滚、糖葫芦
2. **上海** — 生煎（大壶春 / 小杨生煎）、小笼包（南翔）、葱油拌面、排骨年糕、鲜肉月饼、蟹粉小笼
3. **广州** — 肠粉（银记肠粉店 / 八珍煎饺）、艇仔粥、及第粥、云吞面、煲仔饭、碗仔翅
4. **西安** — 樊记肉夹馍、盛志望麻酱酿皮、老白家羊肉泡馍、biang biang 面、油泼面、葫芦头
5. **长沙** — 文和友老长沙、黑色经典臭豆腐、糖油粑粑、剁椒鱼头、米粉、口味虾、湘西腊味
6. **成都** — 钵钵鸡、兔头（双流老妈兔头）、夫妻肺片、甜水面、三大炮、肥肠粉、冒菜
7. **武汉** — 热干面（蔡林记）、豆皮、面窝、精武鸭脖、武昌鱼、排骨藕汤
8. **南京** — 鸭血粉丝汤、盐水鸭（韩复兴）、桂花糖芋苗、赤豆元宵、牛肉锅贴
9. **厦门 / 泉州** — 沙茶面（乌糖 / 四里）、海蛎煎、土笋冻、花生汤、姜母鸭、海蛎炸
10. **潮汕** — 牛肉丸（海记 / 杏花吴记）、手打牛肉丸、粿条、肠粉、蚝烙、白粥配杂咸

## 城市夜市推荐

- **北京** — 簋街（麻辣小龙虾 + 卤煮）、南锣鼓巷（小吃街）、王胖子驴肉火烧
- **上海** — 城隍庙（南翔小笼 + 梨膏糖）、吴江路（小吃一条街）、黄河路（夜宵密集）
- **广州** — 番禺市桥（粤式夜宵）、北京路（大佛寺夜市）、宝业路（粤菜夜宵）
- **西安** — 回民街（羊肉泡馍 + 肉夹馍）、永兴坊（陕西小吃集合）、洒金桥
- **长沙** — 文和友老长沙（小龙虾 + 湘菜）、坡子街（火宫殿）、太平街
- **成都** — 锦里（旅游向）、宽窄巷子（旅游向）、玉林（苍蝇馆子）、建设路
- **武汉** — 户部巷（热干面 + 鸭脖）、吉庆街（夜宵大排档）、万松园
- **南京** — 夫子庙（鸭血粉丝 + 小笼包）、狮子桥（夜宵一条街）、老门东
- **厦门** — 中山路（沙茶面 + 海蛎煎）、曾厝垵、八市（八市是真的本地）

## 时段策略

- **早餐（6:00-10:00）**：生煎、小笼包、热干面、肠粉、豆浆油条、肉夹馍、葱油拌面
- **下午茶（14:00-17:00）**：糖水、双皮奶、煎饼果子、糖葫芦、糖芋苗
- **晚餐（17:00-21:00）**：可作为正餐替代（臭豆腐、麻辣烫、肉夹馍 + 凉皮组合）
- **夜宵（21:00-01:00）**：烧烤、麻辣烫、臭豆腐、米粉、砂锅粥、鸭脖、兔头、口味虾
- **凌晨（01:00-05:00）**：武汉热干面（24h）、广州肠粉、潮汕白粥、兰州拉面（24h）、北京胡辣汤

## 慎选情况

- 商务宴请 → 推粤菜 / 江浙菜 / 鲁菜，小吃太市井
- 对环境要求高（约会、求婚）→ 推西餐 / 日料，小吃摊卫生参差不齐
- 不能吃辣 → 避开湘菜小吃（臭豆腐、口味虾）、川渝小吃（钵钵鸡、兔头）、武汉热干面
- 大量用餐（请 5 人以上吃饭）→ 推正餐，小吃分量小难撑场面
- 严重肠胃不适 / 老人小孩 → 避开生冷、辣度高、街边摊卫生
