# 闽菜知识库

## 十大经典闽菜

1. **佛跳墙** 🦪🦪🦪🦪 — 聚春园（福州东街口）创制，几十种山珍海味同坛
2. **荔枝肉** 🦪 — 福州传统名菜，酸甜挂汁配番茄酱
3. **沙茶面** 🦪 — 厦门四里沙茶面、乌糖沙茶面最有名
4. **海蛎煎** 🦪🦪🦪 — 厦门局口街必吃，地瓜粉 + 蛋
5. **土笋冻** 🦪🦪 — 闽南特色小吃，配蒜泥酱油
6. **福州鱼丸** 🦪🦪 — 鳗鱼丸、鲨鱼丸，连江最正宗
7. **姜母鸭** 🦪 — 厦门特色，老姜 + 番鸭煲
8. **鸡汤氽海蚌** 🦪🦪🦪 — 福州名贵汤，长乐漳港海蚌是上品
9. **武夷熏鹅** 🦪 — 闽北风味，茶香 + 稻秆香
10. **客家盐酒鸡** — 闽西连城，焖出来的古早味

## 城市推荐

- **福州（闽东菜发源地）**
  - 三坊七巷（老字号扎堆，聚春园、安泰楼）
  - 鼓楼 / 鼓东（街坊餐厅，老福州味）
  - 闽侯（鱼丸、桂圆原产地）
  - 闽菜三大分支里福州菜是核心，红糟是灵魂

- **厦门（闽南菜代表）**
  - 中山路 / 局口街（小吃圣地，海蛎煎、土笋冻、沙茶面）
  - 沙坡尾（文艺 + 老厦门味）
  - 临家闽南菜（高端闽南菜代表）
  - 好清香（老牌酒楼）

- **泉州 / 漳州（古早味）**
  - 泉州西街 / 开元寺周边（面线糊、海蛎煎、姜母鸭）
  - 漳州古城（卤面、四果汤）

- **龙岩 / 长汀（闽西客家菜）**
  - 长汀客家菜馆（盐酒鸡、簸箕粄、芋子饺）

- **北京**
  - 闽中大厦、那家老馆（高端闽菜）

## 口味参考

- 🦪 — 鲜度低（冰鲜 / 远途运输）
- 🦪🦪 — 鲜度中（养殖 / 当日上岸）
- 🦪🦪🦪 — 鲜度高（近海当日）
- 🦪🦪🦪🦪 — 鲜度很高（活水 / 现捕现做）
- 🦪🦪🦪🦪🦪 — 鲜度顶（游水 / 刚上岸即烹）

## 慎选情况

- 痛风 / 高尿酸 → 海鲜汤底嘌呤极高，佛跳墙、鸡汤氽海蚌要慎选
- 海鲜过敏 → 几乎所有闽菜核心菜都涉及海产
- 湿热体质 → 沙茶、糟类偏温补，易上火
- 孕期 / 哺乳期 → 生的海产（生蚝、生鱼片）绝对避免
- 北方干冷冬天 → 闽菜偏清淡，暖身效果弱，想吃辣暖身选川菜/湘菜
