# 苏菜知识库

## 十大经典苏菜

1. **松鼠鳜鱼** 🐟🐟🐟 — 苏州松鹤楼原创，形如松鼠，酸甜开胃
2. **蟹粉狮子头** 🐟🐟🐟 — 扬州蟹黄大肉丸，清炖最显功夫
3. **大煮干丝** 🐟🐟 — 淮扬名馔，开国第一宴必上
4. **文思豆腐** 🐟 — 刀工之神作，1 厘米细切 5 千丝
5. **扬州炒饭** 🐟 — 不是随便炒，鸡蛋 + 火腿 + 海参 + 虾仁
6. **蟹黄汤包** 🐟🐟 — 靖江、冶春最有名，先开窗后喝汤
7. **金陵盐水鸭** 🐟🐟 — 桂花鸭是金陵代表
8. **响油鳝糊** 🐟🐟🐟 — 苏州、苏锡帮功夫菜
9. **清炖蟹粉狮子头** 🐟🐟🐟 — 国宴同款，肥而不腻
10. **拆烩鲢鱼头** 🐟🐟 — 淮扬功夫菜，拆骨见功夫

## 城市推荐

- **扬州**
  - 冶春园（淮扬正源，冶春茶社、冶春蒸饺）
  - 富春茶社（三丁包、千层油糕，百年老店）
  - 瘦西湖周边（国宴级菜馆，松鹤楼扬州店）
  - 文昌阁（本地老字号扎堆）

- **苏州**
  - 观前街（苏帮菜起源，松鹤楼、得月楼）
  - 平江路（文人菜氛围，小馆精致）
  - 太湖周边（船菜、湖鲜）
  - 苏锡帮偏甜一点，松鼠鳜鱼最经典

- **南京（金陵）**
  - 夫子庙（秦淮小吃、盐水鸭）
  - 老门东（南京大牌档，集合金陵菜）
  - 颐和路（民国官府菜，民国大菜）

- **上海（海派淮扬）**
  - 人民广场（老吉士、大富贵，海派淮扬）
  - 静安寺（高端淮扬，绿杨村、扬州饭店）
  - 海派淮扬兼容并蓄，吸收了上海本地口味

## 浓淡参考

- 🍃 — 清（如鸡汤、清蒸，刀工菜）
- 🍃🍃 — 偏清（清炖、烩菜，国宴基调）
- 🍃🍃🍃 — 中（红烧、清炒蟹粉，可配米饭）
- 🍃🍃🍃🍃 — 偏浓（焖、煨、狮子头）
- 🍃🍃🍃🍃🍃 — 浓（响油鳝糊、糖醋，下饭神器）

## 慎选情况

- 痛风 / 高尿酸 → 避蟹粉、河豚、浓汤（嘌呤极高）
- 减脂 / 三高 → 避狮子头、响油鳝糊，推清蒸、大煮干丝
- 老人牙口不好 → 避整鱼整鸡，推狮子头、文思豆腐、汤包
- 海鲜过敏 → 避蟹粉、虾仁、河鲀，推盐水鸭、狮子头
- 嗜辣 / 重口味 → 苏菜整体偏淡，建议改推湘菜或川菜
