# 日料知识库

## 十大经典日料

1. **寿司（握り）** 🍣🍣🍣🍣🍣 — 鮨一、鮨的原（上海），鮨匠、然亭（北京）
2. **刺身（お造り）** 🍣🍣🍣🍣 — 厚切 + 萝卜丝 + 紫苏
3. **怀石料理** 🍣🍣🍣 — 一汁三菜，先付 + 八寸 + 椀物 + 烧物
4. **烧鸟（やきとり）** 🍣 — 鸡腿 / 鸡胸 / 鸡胗 / 鸡肝，串烤配盐或酱
5. **天妇罗（天ぷら）** 🍣🍣🍣 — 海老、白身鱼、时蔬，面衣薄而脆
6. **拉面（ラーメン）** 🍣 — 豚骨 / 酱油 / 盐 / 味噌四大流派
7. **鳗鱼饭（うな重）** 🍣🍣🍣🍣 — 关东剖背蒸烤、关西剖腹直烤
8. **亲子丼 / 牛丼** 🍣 — 平民日料，便当首选
9. **寿喜烧（すき焼き）** 🍣🍣🍣 — 牛肉 + 蔬菜 + 寿喜烧汁
10. **抹茶 / 和菓子** — 餐后甜点，季节限定

## 城市推荐

- **东京（终极日料圣地）**
  - 銀座（鮨 さいとう、青柳、小十）
  - 六本木 / 麻布（高端怀石割烹）
  - 新宿思い出横丁（街坊拉面、烧鸟）
  - 筑地 / 丰洲（寿司原点）
  - 参考纪录片《寿司之神》小野二郎数寄屋桥次郎

- **大阪**
  - 道顿堀（平民美食：章鱼烧、大阪烧）
  - 梅田 / 难波（寿司、串カツ）

- **上海（国内日料高地）**
  - 鮨一 / 鮨的原（顶级 Omakase）
  - 割烹篠 / 喜都乃（高端割烹）
  - 南京西路 / 静安（高端日料密集区）
  - 大学路 / 番禺路（平价日料）

- **北京**
  - 鮨匠 / 然亭 / 隐泉（高端寿司）
  - 三里屯 / 工体（居酒屋密集）
  - 五道口（学生向平价日料）

- **广州**
  - 天河 / 珠江新城（高端日料集聚区）
  - 建设六马路（居酒屋一条街）

- **香港**
  - 中环 / 尖沙咀（顶级日料、Sushi Saito）

## 慎选情况

- 海鲜过敏 / 生的不敢吃 → 选烧鸟、寿喜烧、亲子丼
- 不会用筷子 / 不习惯席地而坐 → 选吧台回转寿司或西式日料
- 预算紧张（人均 < 200） → 拉面 / 牛丼 / 回转寿司，别进高端割烹
- 想要吃饱 → 寿司 / 怀石吃不饱，加点套餐或加米饭
- 怕腥 → 避开光物（青鱼），先从白身鱼（比目鱼、鲷鱼）开始
