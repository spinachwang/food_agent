# 徽菜知识库

## 十大经典徽菜

1. **臭鳜鱼** 🧀🧀🧀🧀 — 徽菜头牌，闻臭吃香，腌鳜鱼发酵而成
2. **毛豆腐** 🧀🧀 — 表面长白色绒毛，煎炸后配辣酱
3. **火腿炖甲鱼** 🧀🧀🧀 — 徽商招牌硬菜，火腿 + 山珍 + 甲鱼
4. **胡适一品锅** 🧀🧀🧀 — 徽州待客大菜，多层食材一层层码
5. **徽州毛峰豆腐干** 🧀 — 茶香入味，开胃小菜
6. **黄山炖鸽** 🧀🧀 — 黄山名菜，鸽 + 火腿 + 山药
7. **刀板香** 🧀🧀🧀 — 徽州腊肉，香樟木板焖出来
8. **问政山笋** 🧀 — 徽州春季时令，嫩笋炖肉
9. **石耳炖鸡** 🧀🧀 — 黄山石耳 + 土鸡，滋补
10. **葛粉圆子** — 徽州甜品，葛根粉包馅

## 油度色度参考

- 🧀 — 偏清淡（毛峰豆腐干、问政山笋）
- 🧀🧀 — 适中（毛豆腐、葛粉圆子）
- 🧀🧀🧀 — 偏重（火腿炖甲鱼、刀板香）
- 🧀🧀🧀🧀 — 很重（臭鳜鱼、胡适一品锅）
- 🧀🧀🧀🧀🧀 — 极重（传统徽商待客大菜，传统上桌四层食材）

## 慎选情况

- 怕臭 / 心理预期发酵味 → 臭鳜鱼、毛豆腐都不适合
- 减脂期 / 控制胆固醇 → 重油徽菜不适合
- 痛风 / 高尿酸 → 浓汤炖菜（火腿炖甲鱼）嘌呤高
- 高血压 / 心血管疾病 → 腊味（刀板香）盐分高
- 胃酸过多 / 胃溃疡 → 发酵食物（臭鳜鱼、毛豆腐）刺激
