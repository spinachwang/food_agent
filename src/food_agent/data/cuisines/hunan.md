# 湘菜知识库

## 十大经典湘菜

1. **剁椒鱼头** 🌶️🌶️🌶️ — 湘菜头牌，胖头鱼 + 剁椒蒸，玉楼东最有名
2. **毛氏红烧肉** 🌶️ — 不放酱油用糖色，毛家菜代表
3. **辣椒炒肉** 🌶️🌶️ — 湖南家家会做，下饭神器
4. **口味虾** 🌶️🌶️🌶️ — 长沙夜宵之王，5-9 月最肥
5. **臭豆腐** 🌶️ — 黑色 + 焦壳 + 辣酱 + 蒜汁
6. **腊味合蒸** 🌶️ — 腊肉 + 腊鱼 + 腊鸡，湘西腊味是灵魂
7. **剁椒土豆丝** 🌶️🌶️ — 朴实下饭菜
8. **永州血鸭** 🌶️🌶️🌶️ — 永州经典，鸭血嫩滑
9. **东安鸡** 🌶️🌶️ — 湘菜八大名菜之一，酸辣开胃
10. **糖油粑粑** — 长沙街头甜点，软糯不腻

## 城市推荐

- **长沙（湘菜核心）**
  - 坡子街（火宫殿老牌）
  - 玉楼东（剁椒鱼头源头之一）
  - 五一商圈（湘菜扎堆）
  - 冬瓜山（夜宵一条街，口味虾、烧烤）
  - 太平街（小吃 + 老字号）

- **湘潭 / 韶山（毛家菜发源地）**
  - 韶山毛家饭店（红烧肉、剁椒鱼头）
  - 湘潭农家菜（蛋花汤、湘乡水煮活鱼）

- **湘西（腊味 + 山野）**
  - 凤凰古城（血粑鸭、腊肉）
  - 芙蓉镇（米豆腐、腊味）

- **北京**
  - 湘鄂情（中高端湘菜）
  - 潇湘府（高端商务）
  - 簋街（湘菜夜宵 + 小龙虾）

- **上海**
  - 湘菜一条街集中在普陀 / 静安
  - 谭鸭血、炊烟时代小炒黄牛肉

- 🌶️🌶️ — 中辣（辣椒炒肉、东安鸡，怕辣的人会出汗）
- 🌶️🌶️🌶️ — 重辣（剁椒鱼头、口味虾，推荐嗜辣者）
- 🌶️🌶️🌶️🌶️ — 特辣（永州血鸭、湘西腊味锅）
- 🌶️🌶️🌶️🌶️🌶️ — 魔鬼辣（湖南本地的爆辣卤味）

## 辣型区别（湘菜核心特色）

- **酸辣**：东安鸡、酸辣鱼头，开胃但有醋香
- **香辣**：辣椒炒肉、小炒黄牛肉，辣里带锅气
- **腊辣**：腊味合蒸、湘西腊肉，烟熏 + 辣双重风味
- **咸辣**：剁椒鱼头、豆豉辣椒，下饭为王

## 慎选情况

- 口腔溃疡 / 痔疮发作 → 一律推微辣的毛氏红烧肉、清炒蔬菜
- 孕期 / 哺乳期 → 推温和辣或无辣的米粉、蒸菜
- 小孩 / 老人 → 推番茄炒蛋、蛋花汤、小炒黄牛肉
- 肠胃炎 / 胃溃疡 → 整菜系都不推荐，等恢复再说
- 失眠 / 湿热长痘 → 晚上 8 点后建议避免重辣
