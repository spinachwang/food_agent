# 西餐知识库

## 十大经典西餐

1. **牛排（Steak）** 🥩🥩🥩🥩 — 莫尔顿 / 茹丝葵 / 1515（上海高端）
2. **意面（Pasta）** 🥩🥩 — 那家老牌 / 萨莉亚 / 意轩
3. **鹅肝（Foie Gras）** 🥩🥩🥩 — 福楼、L'Atelier 必点
4. **生蚝（Huitres）** 🥩🥩🥩 — Gillardeau、吉拉多、贝隆
5. **鞑靼牛肉（Steak Tartare）** 🥩🥩🥩🥩 — 法国传统生牛肉
6. **提拉米苏（Tiramisu）** 🥩 — 意餐经典甜品
7. **法式洋葱汤** 🥩 — 巴黎咖啡馆经典
8. **凯撒沙拉** 🥩 — 罗马起源，加凤尾鱼
9. **可颂（Croissant）** 🥩 — 法式早餐必备
10. **惠灵顿牛排** 🥩🥩🥩🥩🥩 — 酥皮 + 蘑菇 + 牛里脊

## 烹饪度 / 复杂度参考

- 🥩 — 简单（沙拉、可颂、洋葱汤）
- 🥩🥩 — 中等（意面、凯撒沙拉）
- 🥩🥩🥩 — 偏复杂（生蚝、鹅肝、提拉米苏）
- 🥩🥩🥩🥩 — 很复杂（牛排、鞑靼牛肉）
- 🥩🥩🥩🥩🥩 — 极复杂（惠灵顿牛排、松露配菜）

## 慎选情况

- 想要吃饱 → 西餐分量小，预算得 1.5-2x 中餐
- 预算紧张（人均 < 200） → 选 bistro 或连锁意餐（萨莉亚），别进米其林
- 不敢吃生（鞑靼 / Medium 以下） → 提前让服务员做 Well Done 或选意面
- 不会用刀叉 / 餐桌礼仪紧张 → 选 casual bistro 而非 fine dining
- 快节奏 / 赶时间 → 西餐节奏慢（2-3 小时常态），选简餐咖啡馆更合适
