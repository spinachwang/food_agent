# 粤菜知识库

## 十大经典粤菜

1. **白切鸡** 🦐🦐🦐🦐 — 清平饭店、广州酒家是源头，配姜葱蒜蓉
2. **烧鹅** 🦐🦐🦐🦐 — 裕记烧鹅、文记壹心鹅，广州鹅最有名
3. **清蒸石斑** 🦐🦐🦐🦐🦐 — 活鱼现杀，姜葱豉油是灵魂
4. **老火靓汤** 🦐🦐 — 椰子鸡、菜干煲猪骨，煲足 3 小时才够味
5. **早茶虾饺** 🦐 — 点都德、陶陶居、广州酒家"一盅三件"
6. **叉烧** 🦐🦐 — 半肥瘦蜜汁叉烧，黯然销魂饭的本体
7. **煲仔饭** 🦐🦐 — 冬天限定，腊味 + 锅巴最香
8. **肠粉** 🦐 — 布拉肠才正宗，豉油要自己浇
9. **糖水** — 杨枝甘露、双皮奶，饭后一碗才圆满
10. **啫啫煲** 🦐🦐 — 猛火生啫，镬气十足，冬夜首选

## 城市推荐

- **广州**
  - 上下九 / 西关（老字号扎堆，广州酒家、陶陶居）
  - 荔湾（街坊茶楼，伍湛记、欧成兴）
  - 天河（高端商务，陶陶居总店、点都德）
  - 海鲜去黄沙水产市场，加工要找老师傅

- **香港**
  - 港岛中环（高端粤菜，利苑、鏞记）
  - 九龙尖沙咀（游客向但水准稳，翠华、添好运）
  - 旺角 / 深水埗（街坊粥面店、菠萝油）
  - 早茶要去莲香楼、陆羽茶室

- **深圳**
  - 福田 / 罗湖（高端粤菜，凤凰楼、利苑）
  - 南山（新兴粤菜融合店）
  - 本地人周末去顺德一日游吃

- **多伦多 / 温哥华（北美）**
  - 多伦多：君临天下、Imperial Court（粤菜高端）
  - 温哥华：Richmond 金钟路、丽晶广场（早茶粤菜一条街）
  - 美国：波士顿龙凤、纽约百乐门

## 鲜度参考

- 🦐 — 一般（冰鲜或死货，街边店水准）
- 🦐🦐 — 中等（普通酒楼活鲜，需指定做法）
- 🦐🦐🦐 — 鲜（明确活水养殖，老饕入门）
- 🦐🦐🦐🦐 — 极鲜（游水现宰，米其林水准）
- 🦐🦐🦐🦐🦐 — 顶级（野生海捕，节令限定）

## 慎选情况

- 痛风 / 高尿酸 → 避海鲜老火汤、避内脏类（猪肚 / 鹅肝）
- 海鲜过敏 → 推烧腊、白切鸡、叉烧、煲仔饭
- 湿热体质（长痘、口苦）→ 避啫啫煲、避煎炸点心
- 脾胃虚寒（容易腹泻）→ 避生鱼片、避冰镇海鲜
- 重口味嗜辣者 → 粤菜偏清淡，建议改推湘菜或川菜
