# 西式快餐知识库

## 主流连锁速查

1. **麦当劳 (McDonald's)** — 全球最大，巨无霸是信仰，1+1=13 穷鬼套餐是打工人的白月光
2. **肯德基 (KFC)** — 中国门店最多，吮指原味鸡 + 蛋挞是招牌，疯狂星期四 V50
3. **汉堡王 (Burger King)** — 真火烤，皇堡是汉堡界天花板，份量大
4. **必胜客 (Pizza Hut)** — 比萨专家，焗饭 / 沙拉 / 披萨一站搞定
5. **达美乐 (Domino's)** — 30 分钟必达，线上订餐体验最好
6. **赛百味 (Subway)** — 健康向，6 寸 1 英尺三明治，健身房首选
7. **塔可贝尔 (Taco Bell)** — 墨西哥卷，美国最爱的平价异国快餐
8. **棒约翰 (Papa John's)** — 披萨第二品牌，披萨胚稍厚
9. **Popeyes** — 炸鸡界黑马，2020 后开始在中国扩张
10. **Five Guys** — 高端汉堡，份量超大但贵

## 城市推荐

- **北京**
  - 国贸 / CBD（午餐高峰店，外卖密集）
  - 五道口（学生向，平价套餐多）
  - 西单 / 王府井（游客向，覆盖早餐到夜宵）
  - 三里屯（汉堡王旗舰店、Popeyes 实验店）

- **上海**
  - 陆家嘴（写字楼密集，午餐高峰长队）
  - 静安寺（24h 麦当劳，凌晨觅食首选）
  - 徐家汇（学生 + 白领混合）
  - 迪士尼小镇（必胜客旗舰店）

- **广州**
  - 天河城 / 太古汇（午餐白领向）
  - 北京路（游客向）
  - 番禺万达（家庭向，开心乐园餐聚集）

- **成都**
  - 春熙路（24h 营业）
  - 太古里（汉堡王旗舰店）

## 时段策略

- **早餐（6:00-10:00）**：麦当劳早餐（6-10 元）、肯德基粥油条（6-15 元）、赛百味早餐三明治
- **午餐（11:00-14:00）**：避峰建议 13:30 后或 11:00 前；午餐套餐 15-25 元
- **下午茶（14:00-17:00）**：甜筒 5 元、蛋挞 6 元、麦乐鸡 10 元三件套
- **晚餐（17:00-21:00）**：套餐时段（30-50 元双人）；疯狂星期四 V50 在这个时段
- **夜宵（21:00-02:00）**：24h 门店覆盖（北上广深均有）；机场 / 高铁站门店

## 慎选情况

- 商务宴请 → 推粤菜 / 江浙菜 / 鲁菜
- 追求仪式感（纪念日 / 求婚）→ 推法餐 / 意大利餐
- 对食材品质要求高（有机 / 现做 / 非冷冻）→ 推精品餐厅
- 厌食症 / 进食障碍患者 → 推家常菜 / 粥面
- 严重过敏（如花生、麸质）→ 必胜客 / 赛百味有过敏原标注，麦当劳和肯德基需逐项确认
