# 中式快餐知识库

## 全国连锁速查

1. **沙县小吃** — 全国门店最多（5000+），拌面 + 蒸饺 + 紫菜蛋花汤三件套 12-15 元
2. **兰州拉面（马艾萨 / 西部马华 / 鑫花溪等）** — 牛肉面为主，15-22 元，毛细到韭叶 5 种面型
3. **杨铭宇黄焖鸡米饭** — 山东济南起家，黄焖鸡米饭 15-25 元，加土豆 / 香菇 / 青菜
4. **桂林米粉 / 柳州螺蛳粉** — 南方强势，南方城市密度高于北方，18-25 元
5. **真功夫** — 蒸菜为主，中式中端 30-40 元，办公区门店密集
6. **吉野家** — 日本品牌，牛肉饭套餐 30-40 元，国内以京沪深为主
7. **嘉和一品** — 京津冀强势，粥 + 包子 + 小菜 15-25 元
8. **东方既白** — 百胜中国旗下（KFC 同公司），中式快餐 25-35 元
9. **杨国福 / 张亮麻辣烫** — 自选式，按重量 / 按荤素，人均 25-40 元
10. **永和大王** — 沪上强势（上海起家），豆浆 + 油条 + 饭团 15-25 元

## 城市推荐

- **北京**
  - 国贸 / CBD（真功夫、吉野家、嘉和一品密集，午餐高峰挤爆）
  - 中关村（沙县、兰州拉面、桂林米粉学生向）
  - 五道口（黄焖鸡米饭扎堆，最便宜）
  - 簋街（夜宵时段 24h 营业的连锁多）

- **上海**
  - 陆家嘴（真功夫 + 吉野家双品牌林立）
  - 徐家汇（学生 + 白领混合，沙县密度高）
  - 静安寺（东方既白总部附近）
  - 浦东机场（24h 连锁覆盖）

- **广州**
  - 天河城（真功夫 + 永和大王集中）
  - 体育中心（兰州拉面 + 黄焖鸡密集）
  - 大学城（沙县 + 桂林米粉学生套餐）

- **深圳**
  - 福田 CBD（吉野家 + 真功夫密集）
  - 南山科技园（黄焖鸡 + 兰州拉面程序员最爱）
  - 龙岗（沙县密度高，平价首选）

## 时段策略

- **早餐（6:00-10:00）**：永和大王（豆浆油条 8-12 元）、嘉和一品（粥 + 包子 10-15 元）、沙县（拌面 + 蒸饺）
- **午餐（11:00-14:00）**：11:30 前 / 13:30 后避峰；12:00-13:00 真功夫 / 吉野家排队最长
- **晚餐（17:00-21:00）**：兰州拉面 / 黄焖鸡晚餐性价比最高，套餐比单点便宜 5-10 元
- **夜宵（21:00-02:00）**：兰州拉面 + 黄焖鸡 + 杨国福是夜宵三巨头，24h 门店覆盖

## 慎选情况

- 商务宴请 → 推粤菜 / 江浙菜 / 鲁菜，环境 + 服务都要跟上
- 家庭聚餐（3 人以上） → 推家常菜馆，连锁快餐座位紧
- 约会 → 推西餐 / 日料 / 创意菜，连锁快餐太"柴米油盐"
- 追求地域特色（想吃正宗川味 / 粤味） → 推对应菜系，连锁快餐都偏标准化
- 严重肠胃不适 → 推白粥 + 小菜（嘉和一品 / 永和大王），避开黄焖鸡 / 麻辣烫
