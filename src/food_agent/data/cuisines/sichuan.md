# 川菜知识库

## 十大经典川菜

1. **麻婆豆腐** 🌶️🌶️🌶️ — 陈麻婆豆腐（成都春熙路总店）是源头
2. **回锅肉** 🌶️ — 蒜苗 + 二刀肉，家庭版就够香
3. **鱼香肉丝** 🌶️🌶️ — 鱼香味是川菜 24 味型代表
4. **水煮鱼** 🌶️🌶️🌶️🌶️ — 重庆做法更地道
5. **夫妻肺片** 🌶️🌶️ — 凉菜下酒神器
6. **担担面** 🌶️🌶️ — 小吃之王，成都甘记最有名
7. **钟水饺** — 甜辣口，成都特色
8. **火锅** — 蜀九香 / 小龙坎 / 大龙燚 / 川西坝子
9. **冒菜** — 单人火锅替代
10. **串串香** — 夜宵首选

## 城市推荐

- **成都**
  - 春熙路（高端商务）
  - 玉林（苍蝇馆子宝藏区）
  - 宽窄巷子（游客向，本地人不去）
  - 海底捞、蜀大侠、小龙坎总部都在这

- **重庆**
  - 解放碑（火锅密集，游客首选）
  - 观音桥（年轻人，本地口碑店多）
  - 南山（夜景火锅）
  - 重庆火锅偏麻辣，比成都更猛烈

- **北京**
  - 簋街（麻辣小龙虾 + 各种川菜）
  - 五道口（学生向平价川菜）
  - 三里屯（高端川菜如许家菜）

- **上海**
  - 云南南路（川菜一条街，老牌）
  - 静安寺（高端川菜如屋里厢）

## 辣度参考

- 🌶️ — 微辣（多数人都能吃）
- 🌶️🌶️ — 中辣（怕辣的人可能会出汗）
- 🌶️🌶️🌶️ — 重辣（推荐给嗜辣者）
- 🌶️🌶️🌶️🌶️ — 特辣（重庆 / 贵州口味）
- 🌶️🌶️🌶️🌶️🌶️ — 魔鬼辣（自家提神用）

## 慎选情况

- 口腔溃疡 / 肠胃不适 → 一律推微辣或非辣川菜（开水白菜、鸡豆花）
- 孕期 / 哺乳期 → 推清淡川菜（开水白菜 + 甜烧白）
- 小孩 → 推酸菜鱼、不辣家常菜
- 老人 → 推粉蒸肉、回锅肉等温和菜
