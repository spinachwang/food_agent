# 浙菜知识库

## 十大经典浙菜

1. **西湖醋鱼** 🐟🐟🐟🐟 — 楼外楼招牌，草鱼 + 醋汁 + 姜末
2. **龙井虾仁** 🐟🐟🐟🐟 — 明前龙井 + 河虾，春天最当时
3. **东坡肉** 🐟🐟 — 慢炖 4 小时，肥而不腻，杭州名片
4. **宋嫂鱼羹** 🐟🐟🐟 — 千年宋嫂鱼羹，醋 + 姜 + 蟹粉
5. **叫花鸡** 🐟🐟 — 黄泥荷叶包裹，江南原味
6. **清汤越鸡** 🐟🐟 — 绍兴名菜，鸡肉嫩白
7. **油焖春笋** 🌱🌱 — 清明前后最嫩
8. **梅干菜烧肉** 🌱🌱🌱 — 绍兴风味，咸香下饭
9. **宁波汤圆** 🌱 — 缸鸭狗最有名
10. **西湖莼菜羹** 🐟 — 莼菜 + 火腿丝，江南水乡味

## 城市推荐

- **杭州**
  - 西湖周边（楼外楼、知味观，杭州名片）
  - 河坊街（南宋御街小吃集合）
  - 龙井村（茶农家菜，明前龙井入菜）
  - 楼外楼、张生记是外地人/本地人都认的

- **宁波**
  - 三江口（宁波菜老本营，缸鸭狗、状元楼）
  - 象山港（海鲜小馆，活鲜最肥）
  - 宁波汤圆、红膏炝蟹、宁波年糕最有名

- **绍兴**
  - 鲁迅故里（咸亨酒店，孔乙己茴香豆）
  - 仓桥直街（梅干菜、霉苋菜、绍兴老酒）
  - 绍兴菜偏咸香，与杭州清淡有别

- **上海（海派浙菜）**
  - 老吉士、张生记上海店（杭帮菜代表）
  - 绿波廊（豫园，融合沪苏浙）
  - 杭州本地菜到上海略加改良

- **多伦多 / 纽约（北美）**
  - 多伦多：龙坤、沪江（杭帮菜 + 江浙融合）
  - 纽约：杭州小笼包、外婆家

## 鲜甜度参考

- 🌱 — 清（如清汤越鸡、龙井虾仁初入口）
- 🌱🌱 — 鲜中带甜（西湖醋鱼、清蒸鱼）
- 🌱🌱🌱 — 鲜甜（中段，宁波汤圆、莼菜羹）
- 🌱🌱🌱🌱 — 偏甜（苏锡浙交界菜，糖醋）
- 🌱🌱🌱🌱🌱 — 甜口（东坡肉烧到位、蜜汁火方）

## 慎选情况

- 痛风 / 高尿酸 → 避河鲜浓汤、避海鲜老汤
- 海鲜过敏 → 避宋嫂鱼羹、龙井虾仁、莼菜羹
- 减脂 / 糖尿病 → 避东坡肉、梅干菜烧肉，推清汤、清蒸
- 老人牙口不好 → 避整鱼整鸡，推龙井虾仁、汤羹类
- 嗜辣 / 重口味 → 浙菜整体偏清淡，建议改推川菜或湘菜
